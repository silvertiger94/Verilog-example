# Verilog-example
블로그에 지식공유를 위한 간단한 베릴로그 코드 예제 모음입니다.
현업에서는 PDK, Library 구현된 부분이 있어 실용성은 떨어질 수 있습니다.
또한, 상용 시뮬레이터로 빌드, 테스트하지 않은 코드입니다.
하드웨어 디자인을 하면서 마주할 수 있는 문제들을 단순한 학습용도로 정리, 구성하였습니다.

# Prerequsite
본 repo는 아래와 같은 환경에서 작성되었습니다. 따라서 아래의 SW, Tool이 필요합니다.
Ubuntu 22.04 LTS
Verilator 
Vivado ML 2023.2
